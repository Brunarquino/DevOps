## [[Fase 1 — Fundamentos de Python]]

**Objetivo:** Conseguir ler, escrever e entender scripts simples usados em automação.
### Conteúdos
- Instalação e ambiente
    - Python 3.x
    - `venv` (ambientes virtuais)
- Sintaxe básica
    - Variáveis e tipos (`int`, `float`, `str`, `bool`)    
    - Operadores
- Estruturas de controle
    - `if / elif / else`
    - `for` e `while`
- Estruturas de dados (muito usadas em DevOps)
    - `list`
    - `dict` (essencial para YAML, JSON, APIs)
    - `set` e `tuple`
- Funções
    - Parâmetros
    - `return`
- Imports e módulos

### Prática DevOps
- Script para:
    - Verificar espaço em disco
    - Listar arquivos grandes em um diretório
    - Ler variáveis de ambiente

## [[Fase 2 — Python para automação de sistemas]]

**Objetivo:** Automatizar tarefas comuns de infra e operações.
### Conteúdos
- Manipulação de arquivos
    - Leitura e escrita de arquivos
    - JSON e YAML
- Tratamento de erros
    - `try / except`
- Execução de comandos do sistema
    - `os`
    - `subprocess`
- Argumentos de linha de comando
    - `sys.argv`
    - `argparse`
        

### Prática DevOps

- Script que:
    - Valida arquivos YAML de Kubernetes
    - Faz backup de diretórios
    - Executa comandos e analisa saída (`kubectl`, `docker`, etc.)

- Infra como Código com Python
    - Pulumi
        
- Manipulação de secrets
    - Variáveis de ambiente

### Prática DevOps

    - Validação de configuração antes de deploy
